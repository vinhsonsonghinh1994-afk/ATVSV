HƯỚNG DẪN CÀI ĐẶT SUPABASE CHO HỆ THỐNG ATVSLĐ VSH
====================================================

── BƯỚC 1: Tạo bảng exam_candidates (danh sách thí sinh) ──

Vào Supabase → SQL Editor → chạy:

CREATE TABLE exam_candidates (
  id bigint generated always as identity primary key,
  full_name text not null,
  password text not null,
  is_active boolean default true,
  created_at timestamptz default now()
);

ALTER TABLE exam_candidates ENABLE ROW LEVEL SECURITY;

CREATE POLICY "allow select" ON exam_candidates
  FOR SELECT USING (true);

── BƯỚC 2: Thêm thí sinh ──

Vào Supabase → Table Editor → exam_candidates → Insert row
Điền: full_name = "Nguyễn Văn A", password = "12345", is_active = true

Hoặc chạy SQL:
INSERT INTO exam_candidates (full_name, password, is_active) VALUES
  ('Nguyễn Văn A', '123456', true),
  ('Trần Thị B',   '123456', true),
  ('Lê Văn C',     '123456', true);

── QUẢN LÝ THÍ SINH ──

Khóa thí sinh (không cho đăng nhập):
UPDATE exam_candidates SET is_active = false WHERE full_name = 'Nguyễn Văn A';

Đổi mật khẩu:
UPDATE exam_candidates SET password = 'matkhaumoi' WHERE full_name = 'Nguyễn Văn A';

Xem kết quả thi:
SELECT * FROM exam_results WHERE exam_id = 'kythi' ORDER BY percent DESC;
